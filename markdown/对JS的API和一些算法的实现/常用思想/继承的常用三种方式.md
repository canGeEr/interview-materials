## 组合式继承
```javascript
function Parent(...args) {
    //调用this绑定实例上的属性和方法
}

function Child(...args) {
    //对父亲构造函数的调用是点睛之笔
    Parent.apply(this, args) // Parent.call(this, ...args) 
    //调用this绑定实例上的属性和方法
}

Child.prototype = new Parent()
Child.prototype.constructor = Child
//优点
//1. 通过Parent.apply调用this，本该在父类实例的引用类型绑定在了子类实例上
//缺点
//2. 由于调用了2次父类的构造方法，会存在一份多余的父类实例属性
```

## 组合寄生式继承
```javascript
function Parent(...args) {
}

function Child(...args) {
    Parent.apply(this, args)
}

Child.prototype = Object.create(Parent.prototype, {
    constructor: {
        value: Child,
        writable: true,
        enumerable: false,
        configurable: true
    }
})
```
